# Air Travel 3-Year Line Chart (1958–1960)

This project visualizes air travel passenger data for 1958, 1959, and 1960 in a single line chart using Tableau.

## How it was created
1. Drag **Measure Names** → Color in Marks panel
2. Drag **Measure Values** → Rows
3. Drag **Month** → Columns
4. Change Marks type → Line
5. Done! Three lines appear, one for each year.
